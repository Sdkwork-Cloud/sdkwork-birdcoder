# SDKWork Skills

This directory stores checked-in skills for sdkwork-birdcoder.

Skills must follow `../sdkwork-specs/SDKWORK_WORKSPACE_SPEC.md` and cite the task-specific standards from `../sdkwork-specs/README.md` instead of copying standard text.

Each real skill must live in a lowercase kebab-case directory with a SKILL.md entrypoint. Skills must not contain product source code, generated SDK output, runtime state, secrets, API keys, private certificates, or user-private files.

## Recommended Skills

- `sdk-generation/`: SDK family generation and verification workflow.
- `app-architecture-check/`: PC architecture boundary verification.
- `component-spec-validation/`: Component spec inventory validation.
- `release-readiness/`: Release and deployment readiness checks.
