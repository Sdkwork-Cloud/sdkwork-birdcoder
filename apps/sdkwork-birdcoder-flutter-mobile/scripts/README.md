# Flutter App Scripts

This directory contains app-specific build, validation, generation, and development utilities for the Flutter application.

## Purpose

App-specific scripts for Flutter builds.

## Allowed Content

- Build scripts
- Validation scripts
- Generation scripts
- Flutter-specific utilities

## Forbidden Content

- Shared workspace scripts (belongs in root `scripts/`)
- Runtime code
- Test files (belongs in `test/`)
