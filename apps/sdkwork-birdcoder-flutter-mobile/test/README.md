# Flutter App Tests

This directory contains application-level integration, widget, and architecture verification tests for the Flutter application.

## Purpose

App-level tests that verify cross-package behavior and architecture boundaries.

## Allowed Content

- Integration tests
- Widget tests
- Architecture verification tests
- Golden tests

## Forbidden Content

- Package-local unit tests (belongs in package `test/` directories)
- Shared test utilities (belongs in root `tests/`)
