# H5 App Scripts

This directory contains app-specific build, validation, generation, release, and development utilities for the H5 application.

## Purpose

App-specific scripts for H5 and Capacitor builds.

## Allowed Content

- Build scripts
- Validation scripts
- Generation scripts
- Release scripts
- Capacitor sync scripts

## Forbidden Content

- Shared workspace scripts (belongs in root `scripts/`)
- Runtime code
- Test files (belongs in `tests/`)
