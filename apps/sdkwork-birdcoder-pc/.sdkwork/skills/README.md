# SDKWork Skills

This directory stores checked-in skills for sdkwork-birdcoder-pc.

Skills must follow `../../sdkwork-specs/SDKWORK_WORKSPACE_SPEC.md` and cite the task-specific standards from `../../sdkwork-specs/README.md` instead of copying standard text.

Each real skill must live in a lowercase kebab-case directory with a SKILL.md entrypoint.
