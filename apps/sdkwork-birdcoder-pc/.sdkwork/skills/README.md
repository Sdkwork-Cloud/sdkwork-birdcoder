# SDKWork Skills

This directory stores only checked-in development-agent skills that are local
to sdkwork-birdcoder-pc. Product AI Skill definitions, installation, lifecycle,
and persistence belong to `sdkwork-skills`; they must not be copied here.

Local agent skills must follow `../../../../../sdkwork-specs/SDKWORK_WORKSPACE_SPEC.md`
and cite task-specific standards from `../../../../../sdkwork-specs/README.md`
instead of copying standard text.

Each real skill must live in a lowercase kebab-case directory with a SKILL.md entrypoint.
