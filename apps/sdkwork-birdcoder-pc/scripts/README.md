# PC App Scripts

This directory contains app-specific build, validation, generation, migration, and development utilities for the PC application.

## Purpose

App-specific scripts that are not shared across surfaces.

## Allowed Content

- Build scripts
- Validation scripts
- Generation scripts
- Migration scripts
- Development utilities

## Forbidden Content

- Shared workspace scripts (belongs in root `scripts/`)
- Runtime code
- Test files (belongs in `tests/`)
