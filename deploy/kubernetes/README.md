# SDKWork BirdCoder Kubernetes Deployment

This directory mirrors the `claw-studio` Helm layout with a chart root, variant values files, and
templated runtime resources.

Base deployment:

```bash
helm upgrade --install sdkwork-birdcoder ./deploy/kubernetes -f deploy/kubernetes/values.yaml
```

Release bundles generated by `scripts/release/package-release-assets.mjs kubernetes` also emit
`values.release.yaml`, which pins the target architecture, accelerator profile, immutable image tag,
and optional image digest used for GitHub release publication.
