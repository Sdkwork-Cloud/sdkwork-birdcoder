## Highlights

- Aligns the BirdCoder workspace onto the `sdkwork-birdcoder-*` package topology modeled after `claw-studio`.
- Keeps BirdCoder positioned as a super AI IDE while standardizing desktop, server, docker, kubernetes, and web release families.
- Adds release smoke evidence, final manifest generation, checksum output, and GitHub release note rendering.

## Scope

- `.github/workflows/ci.yml`
- `.github/workflows/release.yml`
- `.github/workflows/release-reusable.yml`
- `scripts/release/*`
- `deploy/docker/*`
- `deploy/kubernetes/*`
- `docs/*`
