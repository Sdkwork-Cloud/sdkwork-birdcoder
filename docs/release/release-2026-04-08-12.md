## Highlights

- Extends the Step 08 AI CLI launch guard into CodePage and StudioPage run-configuration entry points instead of leaving those surfaces to emit terminal requests blindly.
- Adds a shared run-configuration launch decision in commons so request creation, blocked-launch state, and Settings routing stay aligned across terminal-triggering views.

- Updates Step 08 and architecture docs so the shared guard is now part of the formal terminal standard, not an implementation detail.
