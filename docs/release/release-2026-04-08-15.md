## Highlights

- Extends the unified host runtime descriptor standard into `sdkwork-birdcoder-host-studio`.
- Makes the Studio preview host resolve `global/cn` identity from distribution data instead of host-core defaults.
- Narrows the architecture boundary precisely: `host-studio` may now read `sdkwork-birdcoder-distribution` and `sdkwork-birdcoder-host-core`, but still cannot depend on business packages.
- Expands the shared host runtime contract so Desktop, Host-Studio, and Server now fail together if runtime identity drifts.
