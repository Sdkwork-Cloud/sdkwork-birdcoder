## Highlights

- Adds the first host-studio simulator contract so Simulator now has the same host-derived identity standard already used by Preview.
- Standardizes simulator launch requests and project-scoped simulator evidence persistence on top of the shared run-configuration launch guard.
- Promotes the new simulator contracts into repository release-flow, structure, and architecture gates.
