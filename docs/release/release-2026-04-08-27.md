## Highlights

- Adds replay support to the unified Studio evidence viewer, so a stored Preview, Build, or Simulator evidence entry can now be sent back through the existing terminal execution bus.
- Keeps replay on the same normalized evidence contract instead of creating lane-specific recovery logic.
- Advances Step 07 toward a usable evidence workflow while leaving export and Test lane standardization as the next bounded gaps.
