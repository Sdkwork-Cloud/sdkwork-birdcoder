## Highlights

- Reuses the standardized evidence summary contract directly inside the Studio evidence panel, so diagnostics no longer depend on downloading JSON first.
- Surfaces visible-slice `entries`, `lanes`, `projects`, `profiles`, and `latest launch` metadata beside the existing evidence list.
- Keeps UI diagnostics and export diagnostics on the same summary primitive, avoiding drift between what the panel shows and what the exported bundle contains.
