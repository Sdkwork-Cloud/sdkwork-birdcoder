## Highlights

- Adds a per-entry `Copy Diagnostics` action to the unified Studio evidence viewer so one evidence record can move into triage workflows without leaving the panel.
- Introduces a shared diagnostics bundle builder that emits stable JSON with generation time, lane identity, evidence key, project/run/profile scope, command, and working directory.
- Keeps the enhancement inside the existing Step 07 viewer contract, reusing the same evidence model already used for replay and export.
