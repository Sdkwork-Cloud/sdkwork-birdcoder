## Highlights

- Adds pairwise `Copy Compare Template` to the unified Studio evidence viewer so two selected evidence records can move into root-cause comparison workflows as markdown.
- Introduces a shared comparison-template builder that computes a lightweight summary of lane, project, run-config, profile, command, and working-directory differences.
- Keeps the enhancement local to the Step 07 viewer contract and reuses the same evidence fields already used by replay, export, diagnostics copy, and issue-template copy.
