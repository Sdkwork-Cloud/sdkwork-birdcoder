## Highlights

- Adds `Copy Release Note` to the unified Studio evidence viewer so the current visible evidence slice can move directly into `docs/release`-style markdown.
- Introduces a shared release-note builder that reuses the existing visible-slice summary and evidence keys instead of creating a release-only schema.
- Keeps the iteration inside the Step 07 viewer/export/triage lane and reuses the same clipboard and toast flows already used by diagnostics and issue-template copy.
