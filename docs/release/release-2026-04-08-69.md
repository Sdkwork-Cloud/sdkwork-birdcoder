## Highlights

- Extends Step 10 terminal governance from visible recovery cards into a reusable diagnostics bundle.
- Adds a shared governance diagnostics builder so blocked terminal audit records can be copied as stable JSON without reformatting UI text manually.
- Exposes a `Copy Diagnostics` action inside Terminal governance recovery, reusing the same audit evidence for triage and release collaboration.
