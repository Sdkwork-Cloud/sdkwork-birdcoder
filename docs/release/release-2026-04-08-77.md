## Highlights

- Extends the Step 10 governance regression lane into the unified Studio evidence viewer family.
- Reuses the existing `studio-evidence-viewer` and `studio-evidence-viewer-ui` contracts inside the shared governance regression report instead of creating a separate viewer-only regression pipeline.
