## Highlights

- Synchronizes the multi-engine architecture standard into the executable type source by adding shared engine descriptors, model catalog entries, transport kinds, capability matrices, and canonical coding-session event kinds.
- Extends the data kernel and migration catalog with `model_catalog`, richer `engine_registry` metadata, and transport-aware `coding_session_runtime` columns so storage contracts now match the architecture baseline.
- Updates Step and prompt governance so Step 03, Step 05, Step 09, and the repeat-execution prompt all require the same engine-bridge, canonical-event, and `coding-server core` resource standard.

- Re-issue path: `pnpm release:plan` -> affected types, step docs, prompt, or data-kernel contract files -> `pnpm release:finalize`.
- Writeback targets: `docs/release/releases.json` and `docs/release/release-2026-04-10-01.md`.
