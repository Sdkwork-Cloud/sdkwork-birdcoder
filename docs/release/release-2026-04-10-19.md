## Highlights

- Adds the first Rust `coding-server` projection-read handler skeleton for operation, event, and artifact routes instead of leaving those app-runtime read APIs on `501 not_implemented`.
- Standardizes missing projection reads on the unified `404 not_found` problem envelope so later repository-backed handlers can keep the same error contract.
- Backwrites Step 17, architecture, and the repeat-loop prompt so the next serial closure is fixed on replacing the deterministic host read state with shared provider-backed reads.
