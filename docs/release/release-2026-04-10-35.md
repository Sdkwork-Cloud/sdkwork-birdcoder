## Highlights

- Closes the host-runtime transport loop for default IDE reads: web and desktop now bootstrap with explicit distribution-derived host descriptors, and shell bootstrap injects the shared app/admin client into default workspace/project reads.
- Fixes the HTTP transport URL join so `apiBaseUrl` path prefixes are preserved instead of being discarded.
- Removes module-eager default service freezing from `IDEContext` and `ServiceContext`, making default service creation lazy and runtime-aware.
- Tightens Node-executed workspace package compatibility by adding the missing package export and dependency edges required by the shell-runtime contract tests.
