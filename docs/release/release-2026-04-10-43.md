## Highlights

- Closes the first shared high-level representative app/backend facade on top of the release-backed generated client in `@sdkwork/birdcoder-types`.
- Turns `sdkClients.ts` into a thin transport delegator so representative workspace/project/team/release request assembly no longer lives in infrastructure.
- Adds an executable shared-facade contract and makes it part of `check:release-flow`.
