## Highlights

- Adopts the implemented core projection read facade into the first reusable app-level coding-session detail consumer boundary.
- Adds `loadCodingSessionProjection()` and `useCodingSessionProjection()` for session detail, event, artifact, and checkpoint reads on top of `coreReadService`.
- Locks the consumer slice into `check:release-flow` with a direct Node contract and a shared non-JSX service accessor.

- Rollback entry: `pnpm release:rollback:plan -- --release-tag release-2026-04-11-06 --release-assets-dir artifacts/release`.
