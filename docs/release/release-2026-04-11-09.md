## Highlights

- Closes the typed shared core write facade for `core.createCodingSession` on top of the generated coding-server client.
- Promotes `core.createCodingSession` from the excluded core-governance catalog into the promoted shared-core catalog.
- Wires the first real consumer boundary through `ApiBackedProjectService.createCodingSession()` and preserves refreshed UI session visibility by mirroring the server-created session into local project state.
