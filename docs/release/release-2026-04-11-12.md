## Highlights

- Turns `core.getEngineCapabilities` and `core.listModels` into real Rust-host routes backed by one canonical engine/model catalog.
- Promotes both operations into the shared typed core read facade while keeping only `core.submitApprovalDecision` blocked.
- Wires the first overview consumer path through `loadCodingServerOverview()` and `useCodingServerOverview()` on top of `coreReadService`.
