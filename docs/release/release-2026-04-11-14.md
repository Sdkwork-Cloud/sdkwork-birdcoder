## Highlights

- Closes `17X` by turning `app.listDocuments` into a real Rust-host authority read across demo, legacy sqlite bridge, and direct sqlite provider modes.
- Promotes `listDocuments()` into the shared generated app/admin facade and shared document repository/query/transport boundary.
- Adds `documentService` plus `loadDocuments()` / `useDocuments()` as the first reusable document-facing consumer path and makes both part of release-flow governance.
