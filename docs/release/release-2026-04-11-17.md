- Closes `17ZA` by turning `admin.listDeployments` into a real Rust-host authority read across demo, legacy sqlite bridge, and direct sqlite provider modes.
- Promotes `listAdminDeployments()` into the shared generated app/admin facade while keeping app and admin deployment surfaces explicit.
- Adds `adminDeploymentService` plus `loadAdminDeployments()` / `useAdminDeployments()` as the first reusable admin deployment-facing consumer path and makes both part of release-flow governance.
