## Highlights

- Closes the second Step 18 non-environmental slice by promoting shared engine descriptor, capability, and model-catalog truth into explicit `coding-server` exports.
- Adds a release-blocking contract that prevents `coding-server` from silently rebuilding or default-falling engine catalog truth outside `workbench/kernel.ts`.
- Keeps PostgreSQL live smoke explicitly blocked by environment while moving the next serial closure to Rust-host engine/model parity.

- Observation window: `0` minutes on `pending`.

- Rollback entry: `pnpm release:rollback:plan -- --release-tag release-2026-04-11-20 --release-assets-dir artifacts/release`.

## Notes

- This release closes the TypeScript-side `coding-server` engine truth lane only; Rust host engine/model definitions still remain the next drift-reduction target.
- PostgreSQL live smoke is still blocked when `BIRDCODER_POSTGRESQL_DSN` is absent; this release does not claim live PostgreSQL closure.
- Engine catalog lookup governance now requires explicit `null` for unknown engines instead of hidden fallback behavior.
