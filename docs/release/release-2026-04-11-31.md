## Highlights

- Closes the next Step 06 non-environmental slice by shrinking `CodePage` into an orchestration shell and moving workspace overlays, editor-mode workspace, dialogs, and external terminal integration onto dedicated child components.
- Freezes the new Code page boundary with a release-flow contract so future loops cannot silently re-inline search overlays, editor/diff workspaces, run dialogs, or external terminal integration markup back into `CodePage`.
- Updates bundle-segmentation governance so heavy `chat / editors / run-config` imports remain valid at child-component boundaries instead of forcing them back into the page shell.
