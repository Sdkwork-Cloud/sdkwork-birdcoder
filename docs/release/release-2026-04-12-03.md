## Highlights

- Advances Step 16 `CP16-3` by closing the first provider/UoW-backed persistence slice for Prompt, SkillHub, and AppTemplate evidence.
- Adds `createBirdCoderPromptSkillTemplateEvidenceRepositories()` so `prompt_run`, `prompt_evaluation`, and `app_template_instantiation` all persist through the shared table-repository contract instead of sidecar state.
- Promotes the new evidence-repository contract into `lint`, `check:release-flow`, and governance regression, and realigns the live docs governance baseline to the new 84-check total.

## Notes

- This release closes only the first `CP16-3` persistence slice; coding-server and service consumers still need to adopt the shared repositories before Step 16 is fully closed.
- The governance regression baseline moves from 83 to 84 checks because the new evidence-repository contract is now part of the executable Step 16 closure.
- PostgreSQL live smoke remains environment-blocked outside this change set; this release does not claim a DSN-backed PostgreSQL pass.
