# Examples Directory

## Purpose
Runnable examples, integration examples, sample configs, and SDK/API usage examples.

## Owner
SDKWork Birdcoder team.

## Allowed Content
- Runnable examples
- Integration examples
- Sample configurations
- SDK/API usage examples
- Demo applications
- Tutorial code

## Forbidden Content
- Production code
- Runtime secrets or credentials
- Temporary build artifacts
- Generated SDK output
- Real customer data

## Verification
- [ ] Examples are runnable and documented
- [ ] Examples do not contain secrets or credentials
- [ ] Examples follow SDKWork code style standards
- [ ] Examples have clear usage instructions

## Notes
Examples should be maintained for consumers or verification purposes. They should demonstrate proper SDK/API usage patterns.
